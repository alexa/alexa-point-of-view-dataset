WH-questions

can either be direct or indirect questions that sender requests an action from the recipient. Below are some examples for each. Direct Questions are still questions when standing alone; in contrast, indirect questions donâ€™t make much sense/are not questions/are ungrammatical if you strip off their surrounding text.
  • direct WH-question: Ask John, whose line is it anyway? (the message itself contains questions that are directly identifiable from the sentence with no manipulation.)

  • indirect WH-question: Ask John when he can make it back. (the utterance itself is not a question, but it contains a question.)

    Format can often look like: (Wh-words) + (send, sender pronoun, they) + (helper verb- can, could would should, etc.) + (message containing action or information)
